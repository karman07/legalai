# Custom PDF Viewer

## No Installation Required!

The PDF viewer uses PDF.js loaded from CDN, so no npm packages are needed.

## Features

- Page-by-page PDF rendering on white canvas
- Navigation controls (Previous/Next page)
- Zoom in/out controls
- Rotate page functionality
- Download PDF option
- Clean interface with gray background

Just click "View Document" on any PDF to open the custom viewer!
